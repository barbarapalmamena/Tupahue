# Iglesia Reformada Tupahue

Página web de la Iglesia Reformada Tupahue, Puerto Montt.

## ✅ Páginas

### 1. Página de Inicio (/)
- Hero con imagen de fondo
- Sección de Misión y Visión
- Videos embebidos de YouTube
- Tarjetas de Encuentros
- Sección de Instagram
- Footer completo

### 2. Página de Actividades (/actividades)
- Banner con imagen de fondo
- Calendario de Google embebido

### 3. Página de Nosotros (/nosotros)
- Visión y Misión
- Nuestras Creencias
- Declaración de Fe

### 4. Página de Ministerios (/ministerios)
- Tarjetas de ministerios
- Información de encargados

### 5. Página de Biblioteca (/biblioteca)
- Filtros de categorías
- Tarjetas de libros con reserva
- Conexión a Supabase

### 6. Login (/login)
- Inicio de sesión y registro
- Autenticación con Supabase

## 🎨 Tecnologías
- **Framework:** Next.js
- **Estilos:** CSS Modules
- **Fuentes:** Montserrat, Roboto
- **Iconos:** Bootstrap Icons
- **Base de datos:** Supabase
- **Email:** Resend
- **Deploy:** Vercel
